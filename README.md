# SmartPicker v2.0

SmartPicker 是一个面向 Windows10+ 的智能抽取工具，基于 Python、Tkinter、ttkbootstrap 和拖放功能实现。它可以从多种文件格式中导入数据，并在图形界面中执行随机抽取、范围抽取与多人数抽选，适用于抽奖、分组、学号抽取、点名与名单筛选场景。

---

## 一、项目简介

SmartPicker 旨在提供一个快速、易用的桌面抽取工具：

- 支持 Excel、CSV、TXT、Word 文档导入
- 自动识别有效数据列，并对空值进行清洗
- 提供手动编号范围抽取功能
- 带有抽取动画和结果显示效果
- 支持中文 / 英文界面切换
- 支持可配置抽取人数、结果字体大小、动画速度、重复抽取开关

---

## 二、核心功能

### 2.1 导入多格式数据

- `.xlsx` / `.xls`：自动读取首个有效列，支持常见 Excel 数据表
- `.csv`：自动解析 CSV 文件，并采用 `utf-8` 编码读取
- `.txt`：按行读取文本内容
- `.docx`：优先提取表格内容；如果表格为空，则提取段落文本

### 2.2 自动数据清洗

- 去除空行和空白字符串
- 对 Excel/CSV 中至少包含 2 个非空项的列进行识别
- 将解析结果存入内部候选列表用于抽取

### 2.3 手动范围抽取

- 未导入文件时，可通过“起始编号”和“结束编号”自行生成候选编号列表
- 支持范围最大值保护，超过 50000 条会提示是否继续

### 2.4 多人数抽取与重复控制

- 设置一次抽取人数
- 支持“允许重复抽取”选项
- 默认情况下不允许重复，保证每次抽取项唯一

### 2.5 动画与结果显示

- 抽取时显示“随机滚动中...”的动画效果
- 结果以大字体居中显示，支持自定义字体大小
- 抽取结束时会发出系统提示音

### 2.6 多语言界面

- 提供简体中文与 English 两种界面语言
- 语言可在设置页面中切换，界面文本实时刷新

## 三、环境与依赖

### 3.1 环境要求

- Python 3.10+
- Windows10 以上操作系统

### 3.2 安装依赖

推荐创建虚拟环境后安装依赖：

```bash
python -m venv venv
venv\Scripts\activate
pip install --upgrade pip
pip install ttkbootstrap pandas openpyxl python-docx tkinterdnd2
```

如果不使用虚拟环境：

```bash
pip install ttkbootstrap pandas openpyxl python-docx tkinterdnd2
```

## 五、界面使用说明

### 5.1 导入数据文件

- 点击“📂 导入数据文件”打开文件选择对话框
- 支持直接将文件拖拽到主窗口内导入

### 5.2 数据状态显示

- 导入成功后，窗口顶部会显示文件名称和有效条目数量
- 如果数据无效，会弹出提示框说明问题

### 5.3 设置范围与抽取

- 在“起始编号”与“结束编号”中输入整数
- 若已导入数据，则程序将从导入列表中抽取；若未导入数据，则按范围生成候选编号
- 点击“🚀 开始抽取”启动抽取动画与结果显示

### 5.4 重置数据

- 点击“🗑 重置数据”可清空当前导入结果和界面状态
- 该操作会弹出确认对话框，避免误删

### 5.5 设置面板说明

点击“⚙️ 设置”打开设置窗口，可配置：

- 结果字体大小
- 一次抽取人数
- 抽取动画速度（毫秒）
- 语言切换（简体中文 / English）
- 是否允许重复抽取

## 八、常见问题

### Q1. 导入 Excel 后没有数据？
A1. 请确认 Excel 文件中至少存在一列包含 2 个以上的非空单元格，且该列不是纯空值。
且第一列第一行不能为数字
### Q2. 为什么 Word 文档无法导入？
A2. 目前仅支持 `.docx` 文件，且文件应为 Office Open XML 格式，非老旧 `.doc` 格式。

### Q3. 如何避免重复抽取？
A3. 默认情况下程序不会重复抽取；如果开启“允许重复抽取”，则结果可能出现相同项。
